## 🧠 How the Matrix Works
The Trust Matrix categorizes findings into three pillars:
- **Algorithm Strength:** Evaluating key sizes and curve parameters captured in memory.
- **Protocol Integrity:** Monitoring live negotiations and fallback behaviors.
- **Implementation Context:** Correlating active usage vs. dormant files.

Check out `matrix.json` for our current scoring schema for **NSA CNSA 2.0** compliance.
